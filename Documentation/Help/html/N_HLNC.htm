      NetworkNode3DNode3D, extended with HLNC networking capabilities. This is the most basic networked 3D object. On every network tick, all NetworkNode3D nodes in the scene tree automatically have their network properties updated with the latest data from the server. Then, the special NetworkProcess method is called, which indicates that a network Tick is occurring. Network properties can only update on the server side. For a client to update network properties, they must send client inputs to the server via implementing the INetworkInputHandler interface. The server receives client inputs, can access them via GetInput, and handle them accordingly within NetworkProcess to mutate state.
